# formularioFABLAB

Proyecto FAB LAB INACAP: formulario de registro de visitas con backend Flask y panel administrativo.

## Ejecución rápida

1. Instala dependencias:

`ash
cd "c:\Users\macas\OneDrive\Desktop\fablab formulario"
pip install -r requirements.txt
`

2. Inicia el servidor Flask:

`ash
python fablab_app.py
`

3. En otra terminal, inicia el servidor estático:

`ash
python serve.py
`

4. Abre el formulario en el navegador:

http://localhost:8000/fablab-simple.html

5. Para el panel administrativo:

http://localhost:8000/admin-visitas.html
